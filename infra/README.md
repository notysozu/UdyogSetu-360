# Infrastructure Notes

The `infra/` directory holds deployment-facing placeholders that support the monorepo layout.

- `docker/`: starter Dockerfile templates and notes.
- `nginx/`: reverse proxy placeholders for multi-service routing.
- `mongo/`: init script placeholders.
- `kafka/`: event-bus notes and local setup hints.
- `rabbitmq/`: queue topology notes.
- `n8n/`: runtime notes for workflow execution.
- `scripts/`: helper script placeholders for environment bootstrapping.
